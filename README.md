# README

## Newsstand

A simple enough news stand where user can add in their favorite news publishers' RSS or APIs
and have their own personal newsstand delivered to their browser.

### Tech stack
- Ruby 3.1.0
- Rails 7
- Sidekiq
- Redis
- Postgresql
- Bootstrap
